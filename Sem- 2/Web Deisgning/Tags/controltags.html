Control Type Tags

Anchor Tag: Redirects the page to another link
Target attribute to open the link in new tab
Link For Lab-1
Link for Google

Iframe Tag: Inline frame to embed any document in the current HTML document


List Controls:
List Item Tag: For every item in list(li)
Unordered List: ul tag
  • ABC
  • DEF
  • XYZ
Ordered List: ol tag
  1. ABC
  2. DEF
  3. XYZ

Image Control: img tag

If image not available: Alt attribute


Input Tag: To take input from the user
Label Tag: To specify the information about what to type in the input box

Textarea Tag: For a larger box for input

Already specified inside the text area of input: Placeholder attribute
